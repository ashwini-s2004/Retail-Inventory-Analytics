# Retail Inventory Analytics

## Project Overview

This project analyzes retail inventory data to identify sales trends, low-stock products, supplier performance, and inventory optimization opportunities.

The analysis was performed using Python in Google Colab.

## Business Objectives

* Identify highest-selling product categories
* Detect products approaching reorder levels
* Analyze supplier contribution
* Evaluate inventory performance
* Generate business recommendations

## Tools Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Analysis Performed

### Data Cleaning

* Missing value treatment
* Duplicate record checks
* Stock validation

### Exploratory Data Analysis

* Category-wise product count
* Category-wise sales analysis
* Top-selling products
* Low-stock products
* Supplier inventory distribution

### Visualizations

* Bar Chart
* Pie Chart
* Histogram
* Line Graph
* Correlation Heatmap

## Key Business Insights

1. Identified products requiring immediate replenishment.
2. Determined fast-moving product categories.
3. Highlighted slow-moving inventory.
4. Identified profitable product segments.
5. Evaluated supplier dependency risks.
